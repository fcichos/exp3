    "**Law of Reflection**\n",
    "    \n",
    "If a ray is incident to a reflecting surface under an angle $\\theta_1$ it will be reflected towards under an angle $\\theta_2=\\theta_1$ to the same side of the surface. \n",
    "</div>\n",
    "\n",

    "### Fermat's Principle"
   ]
  },
  {
   "cell_type": "markdown",
   "metadata": {},
   "source": [
    "The law of reflection can be actually obtained from a variational principle saying the light rays propagate along those path on which the propagation time is an extremum. This variational principle is called Fermat's principle.\n",
    "\n",
    "\n",
    "\n",
    "|<img src=\"img/fermat.png\" width=\"60%\">|\n",
    "|-|\n",
    "|**Fig.:** Sketch for deriving the law of reflection from Fermat's principle|\n",
    "\n",
    "Consider now a light ray that should travel from point $A$ to point $C$ via a point $B$ on the mirror surface. In general multiple paths are possible such as the one indicated in the above picture. Clearly this path is not satisfying our reflection law formulated above. Fermat's principle now restricts the path length from $A$ to $C$ to be the one, which takes the least amount of time."
   ]
  },
  {
   "cell_type": "markdown",
   "metadata": {},
   "source": [
    "<div class=\"alert alert-info\">\n",
    "\n",
    "**Fermat's principle**\n",
    "    \n",
    "The path taken by a ray between two given points A, B is the path that can be traversed in the least time.\n",
    "\n",
    "*More precise alternative:* A ray going in a certain particular path has the property that if we make a small change in the ray in any manner whatever, say in the location at which it comes to the mirror, or the shape of the curve, or anything, there will be no first-order change in the time; there will be only a second-order change in the time.\n",
    "</div>"
   ]
  },
  {
   "cell_type": "markdown",
   "metadata": {},
   "source": [
    "So let us consider that contraints to the path length. The total length the light hast to travel via the three points is \n",
    "\n",
    "\\begin{equation}\n",
    "l=l_{1}+l_{2}=\\sqrt{(x-x_1)^2+y_1^2}+\\sqrt{(x_2-x)^2+y_2^2}.\n",
    "\\end{equation}\n",
    "\n",
    "The time that is required by the light to travel that distance $l$ is then given by \n",
    "\n",
    "\\begin{equation}\n",
    "t=\\frac{l}{c},\n",
    "\\end{equation}\n",
    "\n",
    "where $c$ is the speed of light in the medium above the mirror. If this time should now be a minimum, we have to take the derivative of the time $t$ with respect to the position $x$ on the mirror and set that to zero, i.e.,\n",
    "\n",
    "$$\\frac{\\mathrm dt}{\\mathrm dx}=0.$$\n",
    "\n",
    "This results in \n",
    "\\begin{equation}\n",
    "\\frac{x-x_1}{\\sqrt{(x-x_1)^2+y_{1}^2}}=\\frac{x_2-x}{\\sqrt{(x_2-x)^2+y_{2}^2}},\n",
    "\\end{equation}\n",
    "\n",
    "which is actually \n",
    "\n",
    "\\begin{equation}\n",
    "\\frac{x-x_1}{l_1}=\\frac{x_2-x}{l_2}\n",
    "\\end{equation}\n",
    "\n",
    "or \n",
    "\n",
    "\\begin{equation}\n",
    "\\sin(\\theta_1)=\\sin(\\theta_2)\n",
    "\\end{equation}\n",
    "\n",
    "which finally requires \n",
    "\n",
    "$$\\theta_1=\\theta_2$$\n",
    "\n",
    "and is our law of reflection. Thus, refraction (of course) satisfies Fermat's principle.\n"
   ]
  },
  {
   "cell_type": "markdown",
   "metadata": {},
   "source": [
    "While this has been a special example of how to apply Fermat's principle we can define a more general version of it correponding to the following situation also involving an inhomogeneous refractive index $n(\\vec{r})$.\n",
    "\n",
    "|<img src=\"img/fermat_general.png\" width=\"40%\">|\n",
    "|--|\n",
    "|**Fig.:** Sketch for a general description of Fermat's principle. |\n",
    "\n",
    "\n",
    "For this general scenario of light traveling along a path, Fermat's principle is given by\n",
    "\n",
    "\\begin{equation}\n",
    "\\delta \\int\\limits_{A}^{C} n(\\vec{r}) \\mathrm ds=0,\n",
    "\\end{equation}\n",
    "\n",
    "where the $\\delta$ denotes a variation of the path indicated."
